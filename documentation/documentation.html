Overview

Cameo is a flat themed responsive admin template powered by the awesome Twitter bootstrap 3 front-end framework. It comes packed with a plethora of ready to use components and plugins and can be used in the creation of admin dashboards, app backends, Admin panels, CRM or CMS panels. It is clean, fully responsive, well documented and is easy to modify, or style to meet your needs.

You can visit the Bootstrap page to find out all the various components that you can implement with Cameo. The docs are comprehensive and they include everything to help you get started with using all the various components.

Layout Options

  • <body class="app small-sidebar">activates small sidebar sidebar menu
  • <body class="app small-sidebar" data-sidebar="locked">adding data-sidebar="locked" prevents sidebar menu from automatically toggling between small and large sidebar on page load.
  • For boxed layouts, wrap app contents inside <div class="container"></div> . Refer to boxed.html or horizontal_boxed.html for sample.

Top header options

  • add .header-fixed <header class="header header-fixed navbar">to affix header bar to the top on smaller resolutions
  • .off-leftand .off-right classes position left and right header action buttons on lower resolutions. eg
    <a href="#" class="fa fa-bars off-left visible-xs" data-toggle="off-canvas"></a>

Sidebar options

Collapsible sidebar menu HTML structure.
Include slim-scroll plugin to enable scrolling of overflowing content, refer to collapsible.html 

<aside class="sidebar collapsible canvas-left">
    <div class="scroll-menu">

        <!-- begin sidebar main navigation -->
        <nav class="main-navigation slimscroll" data-height="auto" data-size="4px" data-color="#ddd" data-distance="0"></nav>
    </div>

    <!-- begin sidebar footer -->
    <footer></footer>
    <!-- end sidebar footer -->
</aside>
  • To open dropdown menus on hover add .show-on-hover to dropdown container
  • .toggle-sidebar class toggles small sidebar

Off-canvas Navigation

You can setup Off-canvas menus positioned outside of the viewport and slide in when activated.

The markup

Wrap your off canvas element in either .canvas-leftfor a left positioned off-canvas element or .canvas-rightfor a right positioned.

Add your triggers somewhere in your layout. 

<a href="#" class="fa fa-bars off-left visible-xs" data-toggle="off-canvas" data-move="ltr">></a>

<a href="#" class="fa fa-comments-o off-right visible-xs" data-toggle="off-canvas" data-move="rtl"></a>

data-move="rtl" reveals .canvas-right. data-move="lrt" reveals .canvas-left.
